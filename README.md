# Steam Guide Hugo Template

> Derived from DSP_Site (Dyson Sphere Program Guide).
> Copy this entire directory to create a new game guide site.
> Read TEMPLATE_GUIDE.md for full instructions.

## Quick Start

1. Copy `steam-guide-template/` → your new site directory
2. Replace 6 things: hugo.toml, baseof.html, main.css vars, images, materials.toml, content
3. Run `hugo server -p 1616` to preview

See [TEMPLATE_GUIDE.md](TEMPLATE_GUIDE.md) for complete documentation.
