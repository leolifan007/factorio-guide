Space Age introduces interplanetary logistics, quality-tiered items, and four new challenge planets. Your factory no longer fits on one map 鈥?you are building an interplanetary empire.

Big change: Space Age adds quality to items. Machines with quality components run faster, consume less energy, and have larger areas of effect.

Planet Challenge Key Resource

Fulgora Lightning storms, scrap processing Scrap, Holmium

Aquilo Frozen ocean, strict temperature management Cryonite, Ice

Quality Module System

Quality modules add a chance for produced items to be higher quality:

Common (gray) Uncommon (green) Rare (blue) Epic (purple) Legendary (orange)

Strategy: Quality modules are expensive. Prioritize quality on beaconed assemblers and machines whose speed/coverage improvements have the biggest impact.

Interplanetary Logistics

Space Age introduces rocket logistics 鈥?automated rocket launches between planets carrying cargo.

  1. Build rocket silos on each planet
  2. Set up cargo rocket sections in the assembler
  3. Configure logistic requests at the destination silo
  4. Rockets launch automatically when cargo is ready

Tip: Plan your interplanetary supply chain early. Running out of a critical resource on a distant planet is a logistics nightmare.
